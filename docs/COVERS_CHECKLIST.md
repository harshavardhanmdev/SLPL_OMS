# Cover pages: what we have, what is missing

Tracking sheet for book covers in the store. Add new covers by dropping PDFs
in `assets/cover_pdfs/` (any page layout: the batch import crops the front
panel from printed spreads) or by uploading through the admin panel. Either
way the book joins its grade kit automatically.

Last updated: 16 Jul 2026, after the WhatsApp drop (81 new covers imported).
